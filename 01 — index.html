Html stands for Hypertext Makrup lanuage. It gives structure to webpage. It has two important things - Tags and Attributes Tags are special keywords enclosed by angle brackets (< >) that are used to structure and format content on a web page. They are the building blocks of an HTML document, guiding the web browser on how to display text, images, and other media to the user. Most important tags are - <h1> <p> <a> <img> <div> <span> <section> <header> <footer> <nav> <ol> <ul> <li> <input> <button> Attributes are special words used in the opening tag of html element which describe or modify the behaviour and appearance of html elements They always come in name=value pairs using the syntax name="value" Some important attributes - <id class src href alt type name value target> Html element consists of opening tag content and closing tag. It is the builduing block of a webpage.
